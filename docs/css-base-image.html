広大なファンタジー世界を舞台にしたターン制RPGの傑作。プレイヤーは古代の遺産を巡る壮大な旅に出ます。 数十時間に及ぶ重厚なストーリー、戦略的な戦闘システム、そして数百種類に及ぶユニークな装備と魔法があなたを待っています。
